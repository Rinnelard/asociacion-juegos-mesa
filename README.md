# Asociación de Juegos de Mesa

Aplicación web para gestionar una asociación de juegos de mesa con sistema de usuarios, catálogo de juegos, eventos y panel de administración.

## Instalación

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)

## Credenciales de prueba

**Admin:** admin@juegosdemesa.com / admin123  
**Usuario:** usuario@juegosdemesa.com / user123

## Características

- Catálogo de juegos con búsqueda y filtros
- Sistema de reservas
- Valoraciones y comentarios
- Eventos con inscripciones
- Perfil de usuario
- Panel de administración

## Tecnologías

- Next.js 16
- TypeScript
- CSS Modules
- LocalStorage

## Build

```bash
npm run build
```

## GitHub Pages

El sitio está configurado para desplegarse automáticamente en GitHub Pages mediante GitHub Actions.
